# 🎂 Age Calculator (Python)

A simple Python program that calculates a user's age based on their birth year. The program automatically fetches the current year using Python's built-in `datetime` module.

---

## 📌 Features

- Accepts the user's birth year as input.
- Automatically gets the current year.
- Calculates and displays the user's age.
- Uses the `datetime` module instead of hardcoding the current year.

---

## 🛠️ Technologies Used

- Python 3
- `datetime` Module

## 💻 Sample Output

```

```

---

## 🧠 How It Works

1. The user enters their birth year.
2. The program retrieves the current year using `datetime.now().year`.
3. It subtracts the birth year from the current year.
4. The calculated age is displayed.
